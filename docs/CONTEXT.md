# シネマウェディング プロジェクトコンテキスト (CONTEXT.md)

このドキュメントは、「シネマウェディング」LP制作プロジェクトの全体像、デザインルール、およびコーディング規約をまとめたものです。
AIアシスタントに開発を依頼する際は、必ずこのファイルの内容をコンテキストとして読み込ませてください。

## 1. プロジェクト概要とコンセプト

- **プロジェクト名**: シネマウェディング LP
- **コンセプト**: 「映画のような、一生モノのウェディング体験を」
- **キーワード**: 映画的（Cinematic）、上質（Luxury）、情緒的（Emotional）、不変（Timeless）
- **視覚的特徴**: 広い余白、明暗のコントラスト、セリフ体（Serif）を活かしたタイポグラフィによる高級感の演出。
- **アニメーション**: 派手な動きは避け、`fade-in-up` のような「ゆっくりと浮き上がる」上品な動きを基本とする。

## 5. デザインシステム (Tailwind v4 `globals.css` 準拠)

`@theme` にて以下の変数が定義済みです。実装時にはこれらを活用してください。

- **カラーパレット**:
  - `black` (`#1a1a1b`): 映画的な質感の黒。背景やメインテキストに使用。
  - `gray` (`#f9f9f9`): 柔らかい背景用のライトグレー。
  - `accent` (`#b89b72`): ゴールド系。ボタン、アクセント、重要な文字（新規価格の強調など）に使用。
  - `line` (`#06c755`): LINEボタン用のブランドカラー。
- **タイポグラフィ**:
  - `font-serif` (Noto Serif JP): 見出し用。高級感と情緒性を演出。
  - `font-sans` (Sans-serif): 本文用。可読性を重視。
- **アニメーション**:
  - `animate-fade-in-up`: 下から20px浮き上がりながらフェードインする上品なアニメーション（1.5秒・カスタムイージング）。

## 6. Tailwind CSS v4 コーディングルール

コードの可読性と保守性を高め、Linter警告（`suggestCanonicalClasses`）を防ぐため、以下のルールを絶対遵守すること。

1. **推奨（Canonical）クラス名の使用**: 旧バージョンの記法は完全に排除する。
   - 横スクロールやレイアウト保持には `flex-shrink-0` ではなく、必ず **`shrink-0`** または **`grow`** を使用。
2. **Arbitrary values（ブラケット `[]`）の完全禁止**:
   - **アスペクト比**: `aspect-[16/9]` などの記述は禁止。画像・動画枠の比率は **`aspect-16/9`**, **`aspect-4/5`**（SNS縦動画用）, **`aspect-square`**（アイコン用）を直接指定する。
   - **カラー指定**: `text-[#b89b72]` のような直書きは禁止。必ず `text-accent` などのテーマ変数を使用する。
3. **クラスの並び順の自動化**: `prettier-plugin-tailwindcss` のルールに従い、公式推奨の順序にフォーマットする。
4. **コンポーネント分割の基準**: 汎用的なスタイルに `@apply` は使用せず、Reactコンポーネント（`src/components/ui/`）として分割する。
5. **動的なクラス結合の安全確保**: Props等で動的にスタイルを変更する場合は、単なる文字列結合を避け、必ず `src/lib/utils.ts` の `cn` 関数（`clsx` + `tailwind-merge`）を使用する。
6. **モバイルファーストでのレスポンシブ指定**: スマホ向けのスタイルをデフォルトとし、タブレット・PC向けはプレフィックス（`md:`, `lg:` など）で上書きしていく。

## 7. 実装ガイドライン

- **画像の最適化**: Next.jsの `<Image>` コンポーネントを必須とし、必ず適切な `alt` 属性を設定する。
- **セクション分割**: `src/components/sections/` に配置する各セクション（Hero, Concept, Pricing等）は、それ単体でUIが完結するように構築し、`app/page.tsx` でインポートして並べる。
- **ビデオ管理**: `public/videos/` の背景動画は、軽量化されたものをループ再生・ミュート・インライン再生（`autoPlay loop muted playsInline`）で使用する。
